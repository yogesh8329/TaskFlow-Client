# TaskFlow Client

Angular frontend application for authentication and task management.

## Live Demo
https://task-flow-client-seven.vercel.app

## Tech Stack
- Angular
- TypeScript
- RxJS
- JWT Authentication
- Responsive UI
- Vercel Deployment

## Features
- User Registration
- Login
- Forgot Password
- Reset Password
- Protected Routes
- Task CRUD Operations

## Setup
git clone https://github.com/yogesh8329/TaskFlow-Client.git
npm install
ng serve

## Production Build
ng build --configuration production
